Having trouble?

If you're getting access errors, try these steps:

  • Make sure you're on a regular website (not a browser page)
  • Click "Diagnose Page" to check what's wrong
  • Try refreshing the webpage and then the extension
  • Some websites have security restrictions that prevent color changes
